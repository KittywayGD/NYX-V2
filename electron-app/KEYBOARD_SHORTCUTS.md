# ⌨️ NYX-V2 Keyboard Shortcuts

Quick reference for all keyboard shortcuts available in NYX-V2.

## Global Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl + Enter` | Send Message | Envoie le message dans le chat |
| `Ctrl + K` | Focus Input | Place le focus sur l'input de chat |
| `Ctrl + Shift + L` | Clear Chat | Efface tout l'historique du chat (avec confirmation) |
| `Ctrl + Shift + E` | Export Chat | Exporte l'historique en JSON |
| `Ctrl + ,` | Open Settings | Ouvre le panneau des paramètres |
| `Escape` | Close Modal | Ferme les modaux/sandboxes ouverts |

## Chat Interface

- **Send Message**: `Ctrl + Enter` - Envoie votre message à NYX
- **Focus Input**: `Ctrl + K` - Permet de revenir rapidement à l'input
- **Clear History**: `Ctrl + Shift + L` - Efface tout (avec confirmation)
- **Export**: `Ctrl + Shift + E` - Exporte la conversation

## Tips

1. Les raccourcis ne fonctionnent pas quand vous tapez dans un input (sauf `Ctrl + Enter`)
2. Sur macOS, `Ctrl` peut être remplacé par `Cmd` (⌘)
3. Tous les raccourcis sont désactivables dans les paramètres

## Export Formats

L'export du chat supporte 3 formats :
- **JSON** (par défaut) - Format structuré pour traitement
- **Markdown** - Format lisible avec formatage
- **Text** - Format texte simple

Pour changer le format, utilisez le menu dans la barre d'en-tête du chat.
